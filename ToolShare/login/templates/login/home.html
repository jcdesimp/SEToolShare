What is Tool Share? It'll blow your mind.

The product we envision is called Tool Share. At its core, Tool Share is meant to enable neighbors in a community to be able to share items of common use. Anyone wanting to participate can register and be able to share or borrow items immediately. The Tool Share product is intended to improve communities by providing an easy mechanism for sharing items between neighbors.

How much does it cost? Absolutely nothing!

Simply create an account and you can start borrowing tools the same day. If you notice that you have some extra shovels or hammers laying around, then add those tools to your personal tool box and others can borrow your tools you currently don't need.

You can always update tool info. If you want...

Navigate over to your personal tool box under 'My Profile' and change the availability to unavailable. If you don't want to lend out the tool ever again, you can de-register the tool by clicking the 'Remove Tool' button.

Need a tool for a specific time period? Yes I do!

Our reservation system will let you specify the dates that you need a tool for. If the request is approved, when those dates arrive, the tool will automatically be showen as unavailable so you can borrow it for that time period.
